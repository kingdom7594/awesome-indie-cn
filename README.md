# awesome-indie-cn
独立开发者粮仓，记录自己开启独立开发之路过程中收集的资源，也分享给大家能够快速开启自己独立开发之路。主要是面向刚起步阶段的独立开发者或者是起步了一段时间但是没有思路如何去做的同学去分享自己在这个过程中收集的资源。该仓库后续也会持续更新，也欢迎大家分享或者是有问题也请大家帮忙指正。
